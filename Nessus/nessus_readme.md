# Nessus CSV Vulnerability Parser

## 📌 Project Overview
This project is a Python-based Nessus CSV parser designed to analyze vulnerability scan results exported from Nessus.

The script processes the CSV file and extracts key information such as:
- Vulnerability Name
- Risk Level
- CVSS Score
- Affected Hosts

It helps convert raw scan data into structured, readable output for reporting, documentation, or further analysis.

## ▶️ Usage 
1. Export scan results from Nessus to CSV (Assesment.csv)
2. Create a file with the python script (Nessus_parser.py) add the file path (Assesment.csv)
3. Run the Nessus_parser.py file

```python
python3 Nessus_parser.py
```
4. The output will be save in csv (Assesment_parsed.csv)
